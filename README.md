# nvim-k8s-lsp

## 🦶 Requirements

- [yaml-language-server](https://github.com/redhat-developer/yaml-language-server)
- [neovim](https://github.com/neovim/neovim) (0.11 or higher)

## 🍚 Concept

- Schemas detection
- Native CRD matching
- Minimalist

## 👍 Setup

**Lazy**

```lua
return {
  'tonychg/nvim-k8s-lsp',
}
```
